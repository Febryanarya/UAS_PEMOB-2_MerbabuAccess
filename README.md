# merbabuaccess_app

Aplikasi Pendakian Gunung Merbabu - UAS PEMOGRAMAN MOBILE  2

Aplikasi Flutter lengkap untuk manajemen dan pemesanan pendakian Gunung Merbabu dengan integrasi Firebase.

Tentang Proyek

Proyek ini dibuat untuk memenuhi Ujian Akhir Semester Mata Kuliah Pemrograman Mobile 2. MerbabuAccess adalah aplikasi mobile yang menyediakan solusi lengkap untuk pendaki Gunung Merbabu, mulai dari registrasi, pemesanan paket, hingga informasi cuaca dan manajemen profil.

Fitur Utama

1. Autentikasi Pengguna

2. Manajemen Profil

3. Sistem Bantuan & Dukungan

4. Integrasi Firebase
